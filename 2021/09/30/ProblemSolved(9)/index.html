Could not resolve host:mirrorlist.centos.org; Unknown error 的解决方法

发表于2021-09-30|更新于2021-12-16|问题记录
|字数总计:341|阅读时长:1分钟|阅读量:

前言

在给虚拟机安装redis时，选择用命令直接下载安装包，出现了一些问题。

-bash: wget: command not found

下载Redis安装包时，使用命令
wget http://download.redis.io/releases/redis-6.2.5.tar.g

提示 -bash:wget command not found,很明显没有安装wget软件包。一般linux最小化安装时，wget不会默认被安装。

这个问题有两个解决方法：rpm 和yum 安装，选择使用yum安装，更加简单快捷。

输入yum -y install wget，出现错误如下图：

Could not resolve host: mirrorlist.centos.org; Unknown error

本机无法连接网络的网络设置

解决方法：
1.查看安装在本机的网卡，输入nmcli d，ens33 处于disconnected状态；

2.打开网络管理员,输入 nmtui ，选择“Edit a connection”后按回车键（使用TAB键进行选择的选项）


3.选择edit,进行如下配置,（选中Automatically connetc，点击空格）

4.点击OK,依次Back ，回到终端页面。输入systemctl restart network 重启网络服务

5.再次执行yum -y install wget

6.下载Redis,wget http://download.redis.io/releases/redis-6.2.5.tar.g
